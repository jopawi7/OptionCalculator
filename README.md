# OptionCalculator
Our calculator aims to determine precise Option prices based on different models.

To test our project there exists two possibilites:
1) Backend only version. The input can be changed in input.json. Then one should run in the Calculators folder the main function. Finally, it is possible to check the results in the output.json Folder.
2) Full stack version. Start the Backend (see below), open another terminal, start the frontend (see below), open the website as localhost and test the project


# Backend - FastAPI
1) Go in the Backend directory 
2) uvicorn server:app --reload

# Frontend - Angular
1) Go in the Frontend directory
2) ng serve

# Start with Makefile
1) Go to the terminal and type make all
